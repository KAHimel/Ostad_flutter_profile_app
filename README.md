# Flutter Profile Card App

A simple and responsive Flutter Profile App built using basic Flutter widgets and layout system.

This project demonstrates:
- AppBar
- Profile Image
- Name & Bio
- Information Cards
- Social Media Icons
- Responsive Design using MediaQuery
- Dark Mode Support

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/flutter-profile-card.git
```

### 2. Navigate to project folder

```bash
cd flutter-profile-card
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```

---

## 📦 Dependencies

Add these packages in your `pubspec.yaml` file:

```yaml
dependencies:
  flutter:
    sdk: flutter

  font_awesome_flutter: ^10.7.0
  url_launcher: ^6.2.5
```

---

## 🖼️ Assets Setup

Add your profile image inside:

```bash
assets/images/Profile.png
```

Then register assets in `pubspec.yaml`:

```yaml
flutter:
  assets:
    - assets/images/Profile.png
```
